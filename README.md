# Calculadora de IVA Chile (19%)

Una herramienta web rápida, responsiva y sin recargas para calcular el Impuesto al Valor Agregado (IVA) en Chile. Diseñada con un enfoque en la experiencia de usuario (UX) para facilitar cálculos financieros instantáneos.

##  Características Principales

* **Cálculo en Tiempo Real:** Los valores se actualizan automáticamente mientras el usuario escribe, sin necesidad de presionar botones.
* **Escenarios Paralelos:** Muestra simultáneamente el desglose asumiendo que el valor ingresado es Neto y asumiendo que es Bruto.
* **Copiar al Portapapeles:** Botones integrados para copiar los valores limpios (sin formato de moneda) con un solo clic, ideal para facturación y contabilidad.
* **Diseño UI/UX Moderno:** Interfaz limpia, paleta de colores profesional ("Azul Acero") y completamente adaptable a dispositivos móviles (Mobile First).

## Tecnologías Utilizadas

* **HTML5:** Estructura semántica y optimizada para SEO.
* **CSS3:** Variables CSS (Custom Properties), CSS Grid, Flexbox y animaciones suaves. Sin frameworks externos.
* **JavaScript (Vanilla):** Lógica matemática, formateo de moneda nativo (`Intl.NumberFormat`) y manipulación del DOM y la API del portapapeles (`navigator.clipboard`).

##  Uso Legal y Monetización

El proyecto está estructurado como una "Landing Page" completa, incluyendo secciones de contenido SEO educativo sobre el IVA en Chile, y páginas anexas de Política de Privacidad y Términos de Uso, preparándolo para su eventual monetización mediante redes publicitarias.
